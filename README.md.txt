# Proyecto en C++ - Mateo Pardo

## Descripción
Este proyecto fue desarrollado en C++ utilizando programación orientada a objetos y una estructura modular basada en archivos .cpp y .h.

Fue realizado de forma individual como parte de mi formación académica en el Tecnólogo en Informática. El trabajo incluye implementación del sistema, organización por módulos y diagramas de diseño realizados durante el desarrollo.

## Tecnologías utilizadas
- C++
- Programación orientada a objetos
- Makefile

## Estructura del proyecto
- src/: archivos fuente .cpp
- include/: archivos de cabecera .h
- docs/diagramas/: diagramas del sistema y material de apoyo
- Makefile: archivo de compilación

## Compilación
make
